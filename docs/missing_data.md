# Missing data strategy — 2023 National YRBS CSA study

Before imputing anything, we characterized *why* each variable is missing.
Evidence generated by `notebooks/02_missingness.py` (figures in
`reports/figures/`).

## Mechanisms identified

### 1. Questionnaire-version missingness (school-level, ~34–46% blocks)

The 2023 national questionnaire had 107 items: 87 standard items plus 20
new-to-2023 items (social media Q80, current mental health Q84, ACE items
Q99–Q102, school connectedness Q103, parental monitoring Q104, unfair
discipline Q105, difficulty concentrating Q106, English proficiency Q107).
Students in schools sampled for **both** the national and a state/district
YRBS completed only their *site* version (CDC 2023 Overview & Methods, MMWR
2024;73(Suppl-4)), which lacks the national-only items. Consequences visible
in the data:

- Missingness indicators within the new-item block correlate at r = 0.88–0.96;
  correlation with early-questionnaire items is far lower.
- 83–87% of PSUs (school clusters, n ≥ 30 students) are essentially
  **all-or-none** missing on these items — a student-level refusal process
  cannot produce that pattern.
- Response rate *jumps back up* at Q99 (78% answered) after Q98 (54%),
  breaking any monotone-attrition explanation.

**Implication:** for these variables the data are missing by *survey design*,
at the school level, approximately independent of individual student
characteristics. Empirically the received-block and no-block subsamples are
nearly identical on key measures (weighted):

| variable | received block | did not receive |
|---|---|---|
| csa_exposure | 14.8% | 14.1% |
| persistent_sad_hopeless | 40.7% | 37.4% |
| vape_current | 16.9% | 16.5% |
| alcohol_current | 22.3% | 21.7% |
| marijuana_current | 17.1% | 16.7% |
| attempted_suicide_12mo | 9.6% | 9.2% |

**Strategy:** analyze these variables in the **subsample that received them**
(≈ 11,000–13,300 students), keeping survey weights. Do **not** impute a
school-connectedness value for a student who was never shown the question —
listwise deletion *within the subsample* is the honest design-consistent
choice, and the subsample remains large and comparable to the full sample.

### 2. Within-questionnaire attrition (gradual, position-driven)

Response declines gradually with question position among students who
received the national form (e.g., Q48 ≈ 97% answered → Q78 ≈ 65%). This is
survey fatigue / class-period time-out. It is plausibly related to
demographics but not directly to the *values* of unanswered items → treated
as missing at random (MAR) conditional on demographics and design variables.

### 3. Skip-pattern missingness (electronic branching)

Tablet administration skipped follow-up items for students who reported no
engagement in the gateway behavior (e.g., Q30 injurious suicide attempt asked
only of attempters; drinking quantity items only of drinkers). CDC's QN
dichotomies already resolve most of these via logical edits; we use QN
variables, so no action needed beyond using CDC denominators.

### 4. True item nonresponse (sensitive items)

The sexual-violence items carry real refusal/skip: Q19 13.9%, Q20 21.6%
missing. This is the one place where missingness could plausibly relate to
the true value (abuse survivors may decline). We therefore:

- never impute `csa_exposure` (or any exposure/outcome);
- treat exposure prevalence as a likely **lower bound** and say so in the
  paper (survivors declining → underreport bias toward the null);
- run a sensitivity check comparing respondents/nonrespondents on observables.

## Variable-by-variable decisions

| Variable(s) | Missing % | Mechanism | Handling |
|---|---|---|---|
| `csa_exposure` components (Q19–Q21) | 12–33 | attrition + sensitive nonresponse | complete-case; never imputed; underreporting flagged as limitation |
| `household_adversity_score` (Q100–102) | ~35 | questionnaire version (block) | subsample analysis; score requires all 3 items (block co-missing, r ≈ 0.96, so this costs <2% extra) |
| `feel_close_to_school` (Q103), `parent_monitoring` (Q104) | 44–46 | questionnaire version | separate-subsample analysis (n ≈ 11,200 / 10,900); never pooled-imputed |
| `poor_mental_health_30d` (Q84), `social_media_use` (Q80) | 22–24 | questionnaire version | subsample analysis |
| Substance outcomes (QN31–QN55) | 2–29 | attrition + skip patterns | complete-case per outcome (each model keeps rows with that outcome + exposure observed) |
| Demographics (age, sex, grade, race) | <2 | item nonresponse | mode/complete-case; missing-indicator in ML models |
| Mediators (Q26–Q29) | 1–9 | item nonresponse | complete-case in mediation models |

## Modeling consequences

- **Logistic baselines:** complete-case per outcome; adjustment covariates
  with <2% missing use mode imputation; no outcome or exposure imputation.
- **XGBoost:** native NaN handling (missingness becomes information the tree
  can route on) — no imputation needed; we verify SHAP doesn't load heavily
  on design-driven missingness by comparing subsample-restricted fits.
- **No MICE/multiple imputation** for block-missing variables: imputing 44%
  of a variable from a school-level design mechanism adds model-based noise
  without adding information, and risks fabricating within-school variance.
- All prevalence estimates use survey weights; missingness diagnostics showed
  weighted estimates in the received-block subsample match the full sample
  within ~0.7 pp on every checked construct.
